# my-blog-store 
# Full Stack Open 2020
<h3>A React application that stores blog references</h3>
<br/>
<h5><u>Functionalities</u></h5>
<p>Authorised users can Add and Remove blogs </p>
<p>All users can vote and comment on blogs</p>
<br/>
Tech stack :
<br />
<li>React, Redux</li>
<li>NodeJS</li>
<li>MongoDB</li>
<li>Express</li>
